# 유니티로 만드는 ATM (신현주)

필수 구현 사항 완료

선택 구현 사항(통화 단위 적용, 금액 입력시 숫자만 입력, 로그인 기능,)

## <주요기능>

- 로그인 

- 입금

- 출금
